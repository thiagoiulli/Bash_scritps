Some bash executable files used for training shell script.

The most complete file is the [`music_manager`](https://github.com/thiagoiulli/Bash_scritps/blob/master/music_manager), wich is a bash script that organize the musics inside a subfolder according to the mask selected.

## How to run `music_manager`

run `./music_manager /path-to-songs-dir/` and then select the option with the mask that suits you
